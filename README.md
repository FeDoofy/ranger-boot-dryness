# Ranger Boots Dryness Tracker

A RuneLite plugin that tracks how dry you are on ranger boots from medium clue scrolls.

## Features
- Tracks medium Treasure Trail completions automatically via chat messages
- Detects ranger boots drops via collection log notifications
- Counts your Falador guard kills and the medium clues they drop
- Calculates your dryness odds for both boots (1/283.6 per casket) and guard clue drops (1/128, or 1/106 with the medium Falador Diary)
- On-screen overlay and side panel, with one-time calibration from your collection log
